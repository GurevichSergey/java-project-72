# Page Analyzer
[Page Analyzer](https://project72.onrender.com) - a site that analyzes said pages for SEO suitability.

## Requirements

* JDK 17
* Gradle 7
* GNU Make

## Setup

```bash
make setup
```

## Run server

```bash
make start
# Open http://localhost:3000
```
